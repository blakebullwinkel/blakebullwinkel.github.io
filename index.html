Projects

Modeling Electricity Consumption in the US

Employed a multiple linear regression (MLR) model with normal errors to predict household electricity consumption in the US from various residential characteristics.

Used R to conduct:

  • Data cleaning and processing of nationally representative survey data from the US Energy Information Administration
  • Exploratory data analysis of relevant continuous and categorical variables
  • Variable selection using t-tests and partial F-tests, including consideration of possible interaction terms
  • Model diagnostics and transformation of the response variable using a Box-Cox transform
  • Confidence interval calculations for final MLR model coefficients

Early Epidemiological Model Parameters for COVID-19

Modeled early-stage COVID-19 case data in mainland China using a system of ordinary differential equations (SIR model) for senior undergraduate mathematics colloquium.

Project and presentation included:

  • Data cleaning and processing of daily epidemic reports from the Chinese National Health Commission to obtain time series data for susceptible, infected, and removed populations
  • Data and model visualizations using seaborn Python library
  • Model fitting using a Python script that minimizes the sum of squares between the model and the data
  • Parameter extraction and estimation of average infectious period, basic reproduction number, and length of epidemic
  • Epidemic simulation at Williams College using an expanded SIQR model to estimate critical rate of quarantine

Using Linear Programming to Optimize Course Selection

Formulated course selection at Williams as a canonical linear programming (CLP) problem and used Google's OR-Tools Python package to solve by satisfying graduation requirements and user-indicated preferences while minimizing an objective function based on course reviews.

Project and presentation included:

  • Mathematical formulation of problem constraints including scheduling, credit, distribution, and study abroad requirements using binary indicator variables
  • Data processing from multiple sources and parsing into self-defined 'Course' data structure
  • Encoding of contraints listed above using OR-Tools for Python, as well as intended major requirements, course prerequisites, and preclusions
  • Implementation of objective function that minimizes the percentage of course reviewers who "would not recommend the course to a friend"
  • Jupyter Notebook for markdown document and class presentation
