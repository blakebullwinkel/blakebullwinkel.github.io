Wildfire Risk Prediction & Response Optimization in California

I trained tree-based classification models on historical wildfire and weather data to predict the fire risk for a given county and month in California and used mixed-integer programming to determine the optimal assignment of limited firefighters across the state based on total cost.

  • Leveraged minority oversampling (SMOTE) on the training set to address imbalanced classes in the data and double the test set classification accuracy
  • Determined the optimal combination of random forest hyperparameters (number of estimators and minimum samples per leaf node) based on out-of-bag (OOB) error rate
  • Determined the optimal allocation of firefighters in each county and month by using a mixed-integer programming (MIP) solver to encode constraints on the number of available firefighters while minimizing total cost (based on cost of firefighters and fire damages)
  • Evaluated algorithmic bias by comparing the optimized assignment of department-affiliated firefighters to inmate firefighters and estimating the added cost of California's wildfire response this year due to the impact of COVID-19 on the number of inmate firefighters available

Predicting the Outcome of the 2020 Election

I built k-NN and regularized logistic regression models to predict the outcomes of the 2020 presidential and congressional elections using historical, polling, and "fundamentals" data.

  • Incorporated historical election data by deriving a "lagged fold-change" variable, defined as the % democratic / % republican votes in the previous election cycle
  • Used 3-fold cross-validation to tune a k-NN model that predicts fold-change for presidential and congressional elections
  • Built LASSO and ridge-regularized logistic regression models to predict the election outcomes for a given state and year from polling and fundamentals data (GDP, approval rating, etc.)
  • Evaluated model fairness by calculating false positive/negative rates and analyzed possible COVID-19 effects by using unemployment rates as a proxy

DreamDiff Python Package

I worked in a team of three to develop a Python package that implements forward-mode automatic differentiation (AD), root-finding, optimization, and quadratic spline interpolation.

  • Implemented forward-mode AD, which can evaluate and find the derivative of any single or multivariate function with scalar or vector inputs
  • Enabled root-finding using Newton's method and a matplotlib animation method for algorithm visualization
  • Added gradient descent, Nesterov's accelerated gradient descent, and animation methods to find and visualize local minima of functions
  • Implemented quadratic spline interpolation to connect points via piecewise quadratic functions

Analysis of Wildfires, Air Quality, and Public Health

I conducted time-series analysis in R to link spikes in PM2.5 concentration to specific wildfire events in California and used major axis regression to explore correlations between air quality and public health outcomes.

  • Applied smoothing filters (Savitsky-Golay and penalized splines) to remove noise from the PM2.5 concentration time-series
  • Performed seasonal decomposition of time-series by loess (STL) to extract PM2.5 seasonal trends
  • Used major axis regression (variant of OLS) to investigate relationship between the PM2.5 concentration and prevalence of respiratory and cardiovascular diseases
  • Bootstrapped the training data in order to generate 95% confidence intervals for model coefficients

Predicting Agricultural Crop Quality

For the 2020 PepsiCo Data Science Challenge, I built a variety of regression models (MLR, decision tree, random forest) to predict the quality of crops based on agricultural and weather data. I was honored to receive the grand prize among 250+ contest participants.

  • Used R for feature engineering to incorporate time-series weather data into the regression models
  • Incorporated categorical predictors using one-hot encoding
  • Applied a Yeo-Johnson transformation on the response variable to satisfy linear model assumptions
  • Boosted predictive accuracy by tuning the hyperparameters of decision tree and random forest models using 10-fold cross-validation, achieving a final test R2 of 0.9815

Modeling Electricity Consumption in the US

Early Epidemiological Model Parameters for COVID-19

For my senior colloquium at Williams, I modeled early-stage COVID-19 case data in mainland China using a system of ordinary differential equations (SIR model). My project won the 2020 Goldberg Prize for the best mathematics colloquium.

  • Processed data from daily epidemic reports published by the Chinese National Health Commission to obtain time-series data for susceptible, infected, and removed populations
  • Fitted the infected population equation to the time-series using a Python script that minimized the sum of squares between the model and the data
  • Extracted key disease parameters and estimated the average infectious period, basic reproduction number, and length of epidemic
  • Simulated a COVID-19 outbreak at Williams College using an expanded SIQR model to estimate critical rate of quarantine

Using Linear Programming to Optimize Course Selection

I formulated course selection at Williams as a linear programming problem and used Google's OR-Tools Python package to solve by satisfying graduation requirements and user-indicated preferences while minimizing an objective function based on course reviews.

  • Mathematically represented problem constraints including scheduling, credit, distribution, and study abroad requirements using binary indicator variables
  • Processed data from multiple sources and parsed into a self-defined 'Course' data structure
  • Encoded linear contraints using OR-Tools for Python, as well as intended major requirements, course prerequisites, and preclusions
  • Implemented a linear objective function that minimized the percentage of course reviewers who "would not recommend the course to a friend"
